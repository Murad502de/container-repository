# System Design

This folder stores project-level system design documents.

## What Belongs Here

- Project-level or cross-direction component designs.
- Data flow, integration flow, failure, security, and operational design that spans repositories.

## What Does Not Belong Here

- Requirements. Use [../requirements](../requirements/README.md).
- Implementation plans. Use [../technical-design](../technical-design/README.md).
- One-off architecture decisions. Use [../../adr](../../adr/README.md).

Use [system-design-template.md](../../templates/system-design-template.md).
